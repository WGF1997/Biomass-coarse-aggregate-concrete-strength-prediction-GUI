# Biomass coarse aggregate concrete strength prediction GUI

This repository provides an open-source graphical user interface (GUI) for predicting the biomass coarse aggregate concrete compressive strength based on machine learning.

The GUI was developed using Python and Streamlit. Users can input feature values through a web-based interface and obtain the predicted biomass coarse aggregate concrete compressive strength and prediction interval.

## Features

- Web-based graphical user interface
- Machine learning-based prediction
- Real-time output of prediction results
- User-friendly input panel
- Suitable for engineering and academic use
